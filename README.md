# CampusConnect

## Project Description
CampusConnect is a responsive, multi-page web application designed for college students to stay updated on campus events, course information, weather, and important announcements. The app integrates APIs and interactive features to enhance user experience.

## Technologies Used
- HTML5 & Semantic HTML
- CSS3 (with responsive layouts and media queries)
- Bootstrap 5.3
- JavaScript (Vanilla)
- Open-Meteo API (for live weather data)
- Git & GitHub
- GitHub Pages (for deployment)
